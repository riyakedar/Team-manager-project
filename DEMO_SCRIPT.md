# Demo Video Script: Team Task Manager (2-5 Minutes)

## Intro (0:00 - 0:30)
- **Visual**: Show the Login/Signup page.
- **Audio**: "Welcome to Team Task Manager, a professional collaboration platform designed for high-performance teams. Today, we'll look at how roles, projects, and real-time boards work together to streamline your workflow."

## Authentication & Role Selection (0:30 - 1:15)
- **Visual**: Fill out the Signup form, select 'Administrator' for the first user.
- **Audio**: "Starting with a secure signup, we support two roles: Admin and Member. Admins have full control over the project lifecycle, while members focus on execution. Let's sign in as an Admin."

## Creating a Project (1:15 - 1:45)
- **Visual**: Navigate to Projects, click 'New Project', fill 'Website Redesign'.
- **Audio**: "As an Admin, I can create new project workspaces. I'll name this 'Website Redesign'. Once created, the project appears in our directory, ready for the team."

## Inviting Members & Assigning Tasks (1:45 - 2:45)
- **Visual**: Enter project detail. Click 'Invite'. Enter a member email. Create a task 'Design Home Page'.
- **Audio**: "Collaboration starts with people. I'll invite 'Sarah' to this project. Now, let's create our first task on the Kanban board. I'll set it to High priority and set a due date. This UI is built for speed and clarity."

## Real-time Kanban Collaboration (2:45 - 4:00)
- **Visual**: Open the same project in another window (simulated or real). Move the 'Design' task from 'To Do' to 'In Progress'.
- **Audio**: "Here's where it gets powerful. When a team member updates a task status, the board synchronizes instantly across all devices using WebSockets. No refreshing required. Members can easily move their assigned tasks as they progress."

## Dashboard & Conclusion (4:00 - 4:30)
- **Visual**: Show the main Dashboard with updated charts.
- **Audio**: "The dashboard gives me a bird's-eye view of my entire organization: tracking total tasks, identifying what's in progress, and alerting me to overdue items immediately. Secure, real-time, and production-ready. Thanks for watching!"
